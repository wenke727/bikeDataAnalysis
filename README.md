## 数据格式

数据简介：共享单车企业每日订单表
:link: [links](https://opendata.sz.gov.cn/data/dataSet/toDataDetails/29200_00403627)

### 字段介绍
序号	字段名称	字段描述
1	USER_ID	用户id
2	COM_ID	企业 id
3	START_TIME	开始时间
4	START_LNG	开始经度
5	START_LAT	开始纬度
6	END_TIME	结束时间
7	END_LNG	结束经度
8	END_LAT	结束纬度
